# Idukki Collectorate Web Platform

A full-stack web platform developed during my internship 
at Idukki Collectorate, built using the MERN stack. 
The platform aims to digitize and simplify government 
services for citizens and farming communities in Idukki, Kerala.

---

## 🌟 Features

### 🤖 Citizen Chatbot
- Interactive chatbot to assist citizens with common 
  government queries
- Provides instant responses to frequently asked questions
- Reduces workload on government staff

### 🌾 Agriculture Information Portal
- Dedicated portal for farmers in rural communities
- Easy access to farming schemes, weather updates, 
  and agricultural guidelines
- Designed for simplicity so non-technical users 
  can navigate easily

---

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB installed or MongoDB Atlas account

### Installation

# Clone the repository
git clone https://github.com/Aswathy243/agri-web

# Go to project directory
cd agri-web

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Run backend
npm start

# Run frontend
cd ../frontend
npm start
